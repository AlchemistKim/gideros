IOS Game services installation instructions

1) Prerequesites

  • Set up Google Play Services for your app: https://developers.google.com/games/services/ios/quickstart

Things to note:

  • You will need both appID and clientID, note them and don't mix them
  • appID will be set in the bundle as explained in quickstart guide
  • clientID will be set in the GooglePlayServies class inside Plugins/googleplay/GooglePlayService.mm kClientID variable
  • You will need Xcode version 4.5 or higher
  • You will need a mobile device running iOS 6.0 or higher
  • You must set target ios as 5 and higher to compile project

2) Gideros project

  • Create Gideros project
  • Export it as IOS project
  • Import it in Xcode

3) Copying files

  • Copy Plugins/googleplay folder into your exported project Plugins folder
  • Add files to Xcode
  • Follow other described reqesuites, as adding needed frameworks and linker flags and modifying plist
