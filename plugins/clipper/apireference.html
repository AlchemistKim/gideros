API Reference

Polygon

Methods

  • Polygon.new() -- create polygon object
  • Polygon:addPoints(x, y) -- add x and y points to polygon
  • Polygon:getPoints() -- returns table with subtables where 1st element is x and 2nd element is y
  • Polygon:getArea() -- returns area of polygon
  • Polygon:isClockwise() -- return bool if polygon is clockwise
  • Polygon:clean([distance]) -- cleans polygon (CleanPolygon)
  • Polygon:reverse() -- reverses polygon (ReversePolygon)
  • Polygon:simplify([fillType]) -- simplify polygon (SimplifyPolygon)
  • Polygon:getLineOffset(delta [, jointype, endtype, limit]) -- get line offset polygons (OffsetPolyLines)
  • Polygon:getPoligonOffset(delta [, jointype, limit, autoFix]) -- get polygon offset polygons (OffsetPolygons)

Properties

Fill types: (PolyFillType)

  • Polygon.FILL_EVEN_ODD -- even odd fill type
  • Polygon.FILL_NON_ZERO -- non zero fill type
  • Polygon.FILL_POSITIVE -- positive fill type
  • Polygon.FILL_NEGATIVE -- negative fill type

Joint types: (JoinType)

  • Polygon.JOINT_SQUARE -- square joint type
  • Polygon.JOINT_ROUND -- round joint type
  • Polygon.JOINT_MITER -- miter joint type

End types: (EndType)

  • Polygon.END_CLOSED -- closed end type
  • Polygon.END_BUTT -- butt end type
  • Polygon.END_SQUARE -- square end type
  • Polygon.END_ROUND -- round end type

Clipper

Methods

  • Clipper.new() -- create clipper object
  • Clipper:addSubjectPolygon(polygon) -- add subject polygon (PolyType)
  • Clipper:addClipPolygon(polygon) -- add clip polygon (PolyType)
  • Clipper:clear() -- clear poligons
  • Clipper:getBounds() -- returns x, y, width, height of bounds of all provided poligons
  • Clipper:getIntersection([subjFillType, clipFillType]) -- intersection between subject and clip polygons (Execute)
  • Clipper:getUnion([subjFillType, clipFillType]) -- union between subject and clip polygons (Execute)
  • Clipper:getDifference([subjFillType, clipFillType]) -- difference between subject and clip polygons (Execute)
  • Clipper:getExclusion([subjFillType, clipFillType]) -- xor between subject and clip polygons (Execute)
