Bridge Examples

Get native class

local Integer = native.getClass("java.lang.Integer")

Create native object

local i = Integer.new(20)

Call method of native object

print(i:intValue())

Call static method of native class

local sys = native.getClass("java.lang.System")
print(sys.currentTimeMillis()

Convert Gideros path to real

print(native.getPath("|D|image.png"))

Create primitive type array

local boolArray = native.newArray("bool", {true, false, true})

Create array of Objects

local String = native.getClass("java.lang.String")
local objectArray = String.newArray({"test1", "test2", "test3"})

Using proxies for events

local Button = native.getClass("android.widget.Button")
Button:setText("mybutton")
local eventProxy = native.createProxy("android.view.View$OnClickListener", {onClick = function()
	print("Button was clicked")
end})
Button:setOnClickListener(eventProxy)

Accessing current activity and layout for adding new elements

Can be done through GBridge helping class

local bridge = native.getClass("com.giderosmobile.android.plugins.bridge.GBridge")
local activity = bridge.getActivity()
local layout = bridge.getLayout()

Important notes on Native Bridge

  • Currently bridge is Android only and API may differ for IOS systems
  • Bridge is not fool proof, meaning you can find 1000 and one way to crash it, but what works should be stable
  • Strings are both primitive and Object type, you can pass simple string to methods which accept it like "my string" or create String objects and pass them
